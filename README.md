# KampüSepeti - İkinci El Ders Kitabı Pazaryeri

Öğrenciler için ders kitapları, notlar ve kırtasiye malzemeleri alım-satım platformu.

## Teknolojiler
- Frontend: React.js
- Backend: Node.js, Express.js
- Veritabanı: MongoDB
- Authentication: JWT
